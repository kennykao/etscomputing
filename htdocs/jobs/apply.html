Required Documents

In order to apply, you must submit three parts.

  1. A Current Resume

    • Make sure the file name is in the following format last_first_today'sdate_res, i.e. Bear_Oski_08072007_resume

  2. Word Document Exercise

    • Click here to view and download the pdf document
    • Please follow the instructions on the pdf and recreate the entire document using Microsoft Word
    • Save the Word document with the following format: last_first_date_wordex, i.e. Bear_Oski_08072007_wordex

  3. Anticipated Schedule/Availability

    • This can be in pdf, Word, or Excel.
    • Please list all of your classes (including labs and discussions) and all other obligations.
    • Please indicate your preferred hours to work during the week (2-6 hour shifts).
    • Save your availability with the following naming format: last_first_date_avail, i.e. Bear_Oski_08072007_avail
